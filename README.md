# 📋 Sobre o Projeto

O **Parceiro de Obra** é uma plataforma Marketplace moderna desenvolvida para conectar clientes a profissionais da construção civil de forma rápida, segura e eficiente.

A proposta do sistema é facilitar a contratação de profissionais como:

* Pedreiros
* Eletricistas
* Encanadores
* Pintores
* Marceneiros
* Arquitetos
* Engenheiros
* Técnicos especializados

A plataforma oferece uma experiência intuitiva para clientes e profissionais, permitindo divulgação de serviços, busca inteligente, avaliações e gerenciamento completo de contatos.
